

# Code Reviewer Bot 🤖

AI-powered code reviewer built with Streamlit and Gemini API.

🔗 Live: https://code-reviewer-bot-3wfkvzq3jhccc2xvbrtrab.streamlit.app

## Features
- Detects bugs in your code
- Suggests improvements
- Gives quality score out of 10
- Fixes your code automatically
